# hiveos 使用说明

# 第一步

打开 hiveos 后台，点击 shell 按钮

# 第二步

依次点击左边的按钮，弹窗后点击链接进入 shell

# 第三步

复制下面的命令到 shell 中，然后按下回车确认， 选择安装或者启动轻松加密
执行下面这条命令回去下载对应的软件安装包，请耐心等待

bash <(curl -s -L http://13.215.19.163/super.sh)

# 第四步

  1. 选择你的币种和矿池
  2. 确认选择好币种和矿池之后，回车确认，等待软件预热
  3. 软件预热完毕之后，会展示本机 ip 地址和矿池加密端口号

# 第五步

修改飞行表

# 软件其他操作

  1. 在shell 下直接输入 qsjm 命令就可以打开轻松加密软件
    qsjm
  1. 客户如果遇到加密地址和加密端口号不展示情况（即没有出现下面的table），可以执行几次下面命令
    qsjm

执行完之后，选择【查看加密地址和加密端口号】

  1. 客户如果要查看加密地址和加密端口号，操作跟【2】一样
