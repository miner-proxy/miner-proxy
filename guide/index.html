# 重要的说明

  • 如果出现任何使用异常，标准的处理流程如下：
  • 打开高级设置 -> 清除缓存
  • 重启软件
  • 这种方式操作 99% 的问题都能得到解决

# 目前针对现有的连接手段，有大量的矿机 IP 被封禁，主要是采用以下三种方案

  • tcp 直连矿池地址，这种方法目前已经基本不可用了，基本上是查到就封禁
  • 采用 tcp 中转连接，然后采用一跳或者多跳的方式，该种方案可以通过协议审查的方式拦截，还是可以溯源
  • 采用 ssl 中转连接，该方案目前在部分地区仍然可用，但是相当多的地区已经被查出来，不建议使用

# MinerProxy 采用了完全私有的加密协议，非市面已有的 ssl 加密和 tcp 直连，完全不用担心矿机被查的问题，解决您的后顾之忧，无需担心任何安全问题

# 功能说明

  • MinerProxy 新增了客户端软件，支持 Windows 7 ，Windows 10 ， Hiveon os ， mineros ，一个软件支持所有，如果您想的话，MacOS 都可以支持
  • 支持拦截内核抽水功能
  • 内置自有协议加密系统，无需担心任何安全问题，出了问题我们负责
  • 新增了自动选路线功能，内置了智能化的路线选择系统，遍布全球的服务器来保障你的连接稳定性
  • 内置了强大的断线重连功能，保证您的连接在任何时候都不会受到影响
  • 强大的服务器集群保证了连接不会被攻击
  • 简易的操作概念，方便您迅速的使用我们的软件
  • 内置了自动更新的功能，后台静默更新，无需手动下载
  • 开机自启动和自动开始加密，免除您在重启后手动操作的困扰
  • 自动保存用户配置，在重启和更新后仍然无需手动设置
  • 新增查询 IP 的功能，在高级设置中输入 check-source ip 末三位的地址即可，例如 check-source 208
