# В настоящее время большое количество IP-адресов машин для майнинга заблокировано из-за существующих методов подключения, в основном с использованием следующих трех схем

  • прямое подключение по tcp к адресу майнинг-пула, этот метод в настоящее время в основном недоступен, в основном он блокируется при обнаружении
  • Используйте TCP для передачи соединения, а затем используйте один или несколько переходов. Это решение может быть перехвачено просмотром протокола или может быть отслежено до источника.
  • Использование ssl транзитного соединения, это решение все еще доступно в некоторых областях, но многие области были обнаружены, не рекомендуется использовать

# MinerProxy использует полностью закрытый протокол шифрования, несуществующее шифрование ssl и прямое соединение tcp на рынке, поэтому вам не нужно беспокоиться о проверке майнера, решить ваши проблемы и не беспокоиться о любых вопросах безопасности

# Описание функции

  • MinerProxy добавил клиентское программное обеспечение, поддерживает Windows 7, Windows 10, Hiveon os, mineros, одно программное обеспечение поддерживает все, MacOS может поддерживать, если хотите
  • Поддержка перехвата функции подкачки ядра
  • Встроенная собственная система шифрования протоколов, не нужно беспокоиться о проблемах с безопасностью, мы отвечаем за любые проблемы
  • Добавлена ​​функция автоматического выбора маршрута, встроенная интеллектуальная система выбора маршрута и серверы по всему миру для обеспечения стабильности вашего соединения.
  • Встроенная мощная функция отключения и повторного подключения, чтобы гарантировать, что ваше соединение не будет затронуто в любое время
  • Мощный серверный кластер гарантирует, что соединение не будет атаковано
  • Простая концепция работы, удобная для быстрого использования нашего программного обеспечения
  • Встроенная функция автоматического обновления, тихое обновление в фоновом режиме, нет необходимости загружать вручную
  • Автоматический запуск и автоматический запуск шифрования после загрузки, что устраняет проблемы с ручным управлением после перезапуска
  • Автоматически сохранять конфигурацию пользователя, нет необходимости устанавливать вручную после перезапуска и обновления
  • Добавлена ​​функция запроса IP, введите последние три адреса check-source ip в дополнительных настройках, например check-source 208
