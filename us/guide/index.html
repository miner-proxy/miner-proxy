# Currently, a large number of mining machine IPs are banned for the existing connection methods, mainly using the following three schemes

  • tcp direct connection to the mining pool address, this method is basically unavailable at present, basically it is banned when found
  • Use tcp to transfer the connection, and then use one or more hops. This solution can be intercepted by protocol review, or can be traced to the source
  • Using ssl transit connection, this solution is still available in some areas, but a lot of areas have been found out, it is not recommended to use
